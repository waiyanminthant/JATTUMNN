⚙️ JATTUMNN

JATTUMNN is a lightweight Chrome extension that lets you translate any text on the web with a single keyboard shortcut. Just hover over the text you want to translate and press Alt + A (or your chosen shortcut). Use Alt + Q to open the manual translation modal.
